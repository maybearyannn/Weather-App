# Weather-App
A weather app built with React and Vite. Enter a city name to get the current weather using the OpenWeatherMap API.Also use Material UI.

# Getting Started

Install dependencies: npm install

Create a .env file in the project root:

VITE_API_KEY=your_openweathermap_api_key VITE_API_URL=https://api.openweathermap.org/data/2.5/weather

Run the development server: npm run dev

Build for production:

npm run build

# Notes

This project uses OpenWeatherMap for weather data.
